SmartWatch

This watch was built with the most premium materials: CSS & Polymer

<!-- Basic Smartwatch -->
<smart-watch></smart-watch>

<!-- Display an Image -->
<smart-watch img="lol-cat.png"></smart-watch>

Install with bower

$ bower install smart-watch

Or Download the Code

Changing size

You can very easily change the watch size just by changing the font-size property of the element

<!-- Big Watch -->
<smart-watch style="font-size: 30px"></smart-watch>

<!-- Small Watch -->
<smart-watch style="font-size: 5px"></smart-watch>
